Linux (Ubuntu 20.04)
====================

C# fejlesztés
-------------

.NET 5 telepítése:

    $ sudo snap install dotnet-sdk --classic --channel=5.0

A `/snap/dotnet-sdk/current` mappát adjuk hozzá a PATH *elejéhez*:

    export PATH=/snap/dotnet-sdk/current:$PATH

A PATH végéhez hozzáadva nem működött.

Ellenőrzés:

    $ dotnet --version

    $ sudo snap install rider --classic
    $ sudo apt install mono-csharp-shell
